redshiftzero

Predicting police misconduct

This is a data-driven Early Intervention System (EIS) for police departments. The system uses a police department’s officer-level data to predict which officers are likely to have an adverse interaction with the public using historical data and machine learning.

The current code is on GitHub here. I was involved with this project as a postdoc from 2015 to 2016. It is currently deployed in Charlotte-Mecklenberg Police Department (see: wsoctv media coverage). For more, see the KDD 2016 paper “Identifying Police Officers at Risk of Adverse Events” here (PDF).
